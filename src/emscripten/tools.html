Vania Volpe — dev tools

Browser tools deployed with the game. In-game and command-line tools are documented in TOOLS.md.

Asset catalog

Browse every adventure's assets per locale, play audio and animations, and spot locales missing a file the reference locale has.

Generated by tools/gen_asset_catalog.py during make web.

Assets to author

The real art & voice still to make for each adventure, as a checklist with one-click GitHub upload links for the raw files.

Generated by tools/gen_asset_tasks.py during make web; source of truth is each adventure's assets/tasks.json.

Art cost estimate

A tweakable planning estimate for commissioning all of Gina's image assets (backgrounds, props, animation frames) from a professional artist.

Self-contained page; defaults seeded from each adventure's assets/tasks.json.

Walk-mask editor

Paint a scene's walkable area over its background and export a walkable.walk to commit. Browser counterpart of the in-game paint mode (debug layer → W).

Format spec in MOVEMENT.md; loaded by walk_grid_init.

In-game debug layer

Press D in any build: hotspot/POI/walk-grid overlay, the rect picker (drag prints an SDL_Rect), and the walk-mask paint mode (W to toggle, S to save — native builds).
